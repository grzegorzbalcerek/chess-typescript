chess-typescript
================

The game of Chess written in TypeScript.

Compilation:

```
tsc *.ts -t ES2020 -m commonjs
```

Tests:

```
node color.js
node figure.js
node field.js
node move.js
node figuremoves.js
```

The suggested order of reading:

 * color.ts
 * figure.ts
 * field.elm
 * move.ts
 * figuremoves.ts
