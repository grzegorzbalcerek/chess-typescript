chess-typescript
================

The game of Chess written in TypeScript.

tsc *.ts -t ES2020 -m commonjs
node color.js
node figure.js
node field.js

The suggested order of reading:

    color.ts
    figure.ts
    Field.elm
    Move.elm
    FigureMoves.elm
    Board.elm
    Game.elm
    Rank.elm
    ComputerPlayer.elm
    Chess.elm
    Util.elm
